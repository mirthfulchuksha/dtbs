Get Started Try

Striking

Arrange schemas in a visually striking D3 powered format, or utilize a more traditional view for cogent data organization. DTBS facilitates an engaging, interactive experience during the potentially tedious schema-building process

Productive

DTBS supports both relational and non-relational databases, allowing users to design and save schemas for virtually any data set. Build a schema in either SQL or Mongo, and toggle between various ORMs for intuitive schema building

Intuitive

Clean and simple interface makes creating layouts easy. Auto-generate a code base by starting with schema visualization, or edit and submit code to build the visuals instantly. Download the code and a PNG of the layout

How it Works



SQL


Build a relational table-based layout using the SQL view of the application. Start with a single table and its required primary key value, then continue to build complex relationships with other tables using the foreign key feature. DTBS supports various data types for field creation, making complicated data simple to visualize

Mongo


When building a Mongo schema, create at least one key-value pair. In addition to simple data types, DTBS enables the use of nested schema documents, up to ten levels deep. View the data in a dynamic tree-like structure, or toggle views to see a static dendrogram representation. View generated database code in either raw Mongo or parsed Mongoose
